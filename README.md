# 本项目使用 [yarn](https://yarn.bootcss.com/) 管理依赖包

## 步骤

    pull项目后

    yarn install 运行安装依赖包
    进入项目根目录
    yarn start 即可运行
    yarn run build 打包

## 项目依赖

    react
    react-router-dom
    axios
    antd
